# HR Analytics Dashboard 📊

## Project Overview
This project focuses on analyzing employee attrition trends, workforce demographics, and HR performance metrics using Power BI.

The dashboard helps understand:
- Employee Attrition Rate
- Department-wise Analysis
- Age Group Distribution
- Salary Insights
- Job Role Performance
- Work-Life Balance Impact

## Dataset Information
- Records: 1480 Employees
- Features: 38 Columns
- Domain: Human Resources Analytics

## Tools Used
- Power BI
- Excel / CSV
- Data Cleaning
- DAX
- Data Visualization

## Key KPIs
✔ Total Employees  
✔ Attrition Count  
✔ Attrition Rate  
✔ Average Age  
✔ Average Salary  
✔ Department Distribution

## Files Included
- HR_Analytics.csv
- Power BI Dashboard (.pbix)
- Dashboard Screenshots

## Project Goal
To identify factors affecting employee attrition and provide business insights for HR decision-making.
